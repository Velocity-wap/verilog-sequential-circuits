
A collection of sequential logic circuits implemented in Verilog HDL from scratch. Covers shift registers and counters as part of a self-directed study in digital design and RTL development.
Modules:

  - SISO Shift Register — Serial in, serial out, 8-bit
  - SIPO Shift Register — Serial in, parallel out, 8-bit
  - Parallel Load Register — Load data in parallel, shift serially
  - Up-Down Counter — Synchronous counter with direction control

Tools: Verilog, VSCode, GTKWave
